# Practice 3
**FileReader** - класс, предназначенный для чтения файлов, состоящий из следующих методов:

1. *readLine* - возвращает строку, состоящую из имени файла, номера строки и строки, взятой из файла
2. *hasMoreLines* - проверяет, прочитаны ли все строки

**Example** - класс, содержащий следующий метод:
1. *getLongestLineCharsCount* - возвращает количество символов в самой длинной строке, принимая в качестве параметра экземпляр класса FileReader.
