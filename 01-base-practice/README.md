# Git & Java. Time Tracking

Данный проект предназначен для вычисления расстояния между точками на плоскости. 

Проект содержит следующие классы:

Point - класс точка, у которого есть приватные поля x, y (координаты), а также методы getX и getY для получения координат;
Segment - класс отрезков, у которого есть приватные поля point1, point2 (точки), а также метод getSegmentLength, предназначенный для получения расстояния между 2-я точками.

# Как запустить проект?

1. Перейти в папку src/main/it/sevenbits
2. Выполнить команду javac Main.java
3. Выполнить команду java Main
